中文說明：
這裡附上已經編譯過個 Marble Run 的 Android 程式。

安裝 Marble Run：
1. 請將 marble_run.apk 放入到手機裡的SD卡上。
2. 在 Google Play 上，尋找 檔案總管 的 Android 程式。建議：ASTRO File Explorer
3. 安裝 檔案總管 的程式之後，請在 Android 裡瀏覽到 marble_run.apk 的位置。
4. 執行並且安裝 marble_run.apk。

使用說明：
1. 該遊戲使用 Android 手機的三軸感應器。請轉動手機成橫狀，再使用雙手握住手機兩側的邊緣。
2. 遊戲操作說明在於第一關、第二關、及第七關會出現。
3. 你是紅色的球，目標是想但法到達黑洞裡 (終點)。請勿碰到黑色的無底洞，否則會輸掉。
4. 將手機上向上甩即可讓球跳起來。

English README:
The Android program is included in this folder.

Installation:
1. Put the "marble_run.apk" onto your phone's SD card.
2. Find any File Explorer programs on Google Play. Recommended: ASTRO File Explorer.
3. After installing the file explorer, browse to where the "marble_run.apk" was placed in.
4. Run and install.

How to play:
1. This game uses the Android phone's accelerometer. Please rotate the phone until it's held in a
   reverse landscape position, then hold the phone on the sides with both hands.
2. There are tutorials in Stage 1, Stage 2, and Stage 7.
3. You are the red ball in control, you goal is to reach the black hole at the end (destination).
   Do not touch the Void, else you will lose.
4. Flick the phone with an upward motion to make the ball jump.
